Visual Studio Code Live Server Extension

Installation

This first step is to Install the Live Server Extension for Visual Studio Code

  1. In the Visual Studio Code, click the Extension icon on the lft hand vertical bar,
  2. In the Search box: write Live Server,
  3. You should see the extension page appear on the right; click install.
You have installed the Live Server Extension for Visual Studio Code!

Using Live Server

  • Open a project to Go Live from the status bar to turn the server on/off
  • Right click on a HTML file from Explorer Window and click on Open with Live Server
